# Contributing to Curated LLM APIs

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** this repository
2. **Edit** `README.md` following the existing format
3. **Submit** a pull request

### Entry Format

Each entry should follow this format:

```markdown
- [Provider Name](https://provider-url.com) - Brief description of the service.
```

### Requirements

- Real, active service with a working URL
- Accurate, concise description (1-2 sentences)
- Correct category, alphabetical order, no duplicates

### Updating Pricing

1. Include the date you verified the pricing
2. Link to the official pricing page
3. Use per 1M tokens as the standard unit

## Guidelines

- Be respectful and constructive
- One provider per pull request
- Keep descriptions neutral and factual
- Test all links before submitting
